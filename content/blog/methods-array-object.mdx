Understanding how to work with arrays and objects is fundamental in JavaScript. Both data structures come with a variety of methods that allow you to manipulate and interact with data efficiently. In this article, we'll explore some essential methods for arrays and objects, including practical examples to illustrate their usage.

## Array Methods

### 1. `.push()`

The `.push()` method adds one or more elements to the end of an array and returns the new length of the array.

#### Example:

```js showLineNumbers
const numbers = [1, 2, 3];
const newLength = numbers.push(4, 5);
console.log(newLength); // Outputs: 5
console.log(numbers); // Outputs: [1, 2, 3, 4, 5]
```

### 2. `.pop()`

The `.pop()` method removes the last element from an array and returns that element. This method changes the length of the array.

#### Example:

```js showLineNumbers
const numbers = [1, 2, 3, 4];
const lastElement = numbers.pop();
console.log(lastElement); // Outputs: 4
console.log(numbers); // Outputs: [1, 2, 3]
```

### 3. `.shift()`

The `.shift()` method removes the first element from an array and returns that element. It changes the length of the array.

#### Example:

```js showLineNumbers
const numbers = [1, 2, 3, 4];
const firstElement = numbers.shift();
console.log(firstElement); // Outputs: 1
console.log(numbers); // Outputs: [2, 3, 4]
```

### 4. `.unshift()`

The `.unshift()` method adds one or more elements to the beginning of an array and returns the new length of the array.

#### Example:

```js showLineNumbers
const numbers = [2, 3, 4];
const newLength = numbers.unshift(1);
console.log(newLength); // Outputs: 4
console.log(numbers); // Outputs: [1, 2, 3, 4]
```

### 5. `.map()`

The `.map()` method creates a new array populated with the results of calling a provided function on every element in the calling array.

#### Example:

```js showLineNumbers
const numbers = [1, 2, 3];
const doubled = numbers.map((num) => num * 2);
console.log(doubled); // Outputs: [2, 4, 6]
```

### 6. `.filter()`

The `.filter()` method creates a new array with all elements that pass the test implemented by the provided function.

#### Example:

```js showLineNumbers
const numbers = [1, 2, 3, 4, 5];
const evenNumbers = numbers.filter((num) => num % 2 === 0);
console.log(evenNumbers); // Outputs: [2, 4]
```

### 7. `.reduce()`

The `.reduce()` method executes a reducer function (that you provide) on each element of the array, resulting in a single output value.

#### Example:

```js showLineNumbers
const numbers = [1, 2, 3, 4];
const sum = numbers.reduce((accumulator, current) => accumulator + current, 0);
console.log(sum); // Outputs: 10
```

### 8. `.sort()`

The `.sort()` method sorts the elements of an array in place and returns the sorted array. By default, it sorts the elements as strings in ascending order. For numerical sorting or custom sorting, you need to provide a comparison function.

#### Default Sorting (Alphabetical)

```js showLineNumbers
const fruits = ["banana", "apple", "cherry"];
fruits.sort();
console.log(fruits); // Outputs: ["apple", "banana", "cherry"]
```

#### Numerical Sorting

To sort numbers, provide a comparison function to `.sort()`:

```js showLineNumbers
const numbers = [4, 2, 5, 1, 3];
numbers.sort((a, b) => a - b);
console.log(numbers); // Outputs: [1, 2, 3, 4, 5]
```

#### Descending Order

To sort numbers in descending order:

```js showLineNumbers
const numbers = [4, 2, 5, 1, 3];
numbers.sort((a, b) => b - a);
console.log(numbers); // Outputs: [5, 4, 3, 2, 1]
```

### 9. `.forEach()`

The `.forEach()` method executes a provided function once for each array element.

#### Example:

```js showLineNumbers
const numbers = [1, 2, 3];
numbers.forEach((num) => console.log(num));
// Outputs:
// 1
// 2
// 3
```

### 10. `entries()`

The `entries()` method returns a new Array Iterator object that contains the key/value pairs for each index in the array.

#### example

```js showLineNumbers
const numbers = ["a", "b", "c"];
const iterator = numbers.entries();
console.log([...iterator]); // Outputs [[0,"a"],[0,"b"],[0,"c"]]

for (const [index, value] of iterator) {
  console.log(index, value);
}
// Outputs:
// 0 "a"
// 1 "b"
// 2 "c"
```

### 11. `from()`

The `from()` method creates a new, shallow-copied Array instance from an array-like or iterable object.

#### Example 1: Converting a Set to an Array

```js showLineNumbers
const set = new Set(["a", "b", "c"]);
const arrayFromSet = Array.from(set);
console.log(arrayFromSet); // Outputs: ["a", "b", "c"]
```

#### Example 2: Converting a String to an Array of Characters

```js showLineNumbers
const str = "hello";
const arrayFromString = Array.from(str);
console.log(arrayFromString); // Outputs: ["h", "e", "l", "l", "o"]
```

#### Example 3: Using a Map to Create an Array

```js showLineNumbers
const map = new Map([
  ["key1", "value1"],
  ["key2", "value2"],
]);
const arrayFromMap = Array.from(map);
console.log(arrayFromMap); // Outputs: [["key1", "value1"], ["key2", "value2"]]
```

#### Example 4: Applying a Mapping Function with `Array.from()`

```js showLineNumbers
const numbers = [1, 2, 3, 4];
const doubled = Array.from(numbers, (num) => num * 2);
console.log(doubled); // Outputs: [2, 4, 6, 8]
```

#### Example 5: Converting an Array-like Object with Gaps to an Array

```js showLineNumbers
const arrayLike = { 0: "a", 2: "b", length: 3 };
const arrayFromArrayLike = Array.from(arrayLike);
console.log(arrayFromArrayLike); // Outputs: ["a", undefined, "b"]
```

#### Example 6: Creating an Array with `Array.from()` and a Generator

```js showLineNumbers
const range = (start, end) =>
  Array.from({ length: end - start }, (_, i) => i + start);
console.log(range(1, 5)); // Outputs: [1, 2, 3, 4]
```

### 12. `fill()`

The `fill()` method fills all the elements of an array from a start index to an end index with a static value.

#### Example:

```js showLineNumbers
const numbers = [1, 2, 3, 4];
numbers.fill(0, 2, 4);
console.log(numbers); // Outputs: [1, 2, 0, 0]
```

### Using `for...of` with Arrays

The `for...of` loop is a modern way to iterate over the elements of an array.

```js showLineNumbers
const numbers = [1, 2, 3];

for (const number of numbers) {
  console.log(number);
}
// Outputs:
// 1
// 2
// 3
```

```js showLineNumbers
["a", "b"].concat(["c"]); // ["a","b", "c"]

["a", "b"].join(["-"]); // "a-b"

["a", "b", "c"].slice(1); // ["b", "c"] --> arr.slice(start, end)
```

## Object Methods

### 1. `.keys()`

The `Object.keys()` method returns an array of a given object's own enumerable property names.

#### Example:

```js showLineNumbers
const person = { name: "John", age: 30 };
const keys = Object.keys(person);
console.log(keys); // Outputs: ["name", "age"]
```

### 2. `.values()`

The `Object.values()` method returns an array of a given object's own enumerable property values.

#### Example:

```js showLineNumbers
const person = { name: "John", age: 30 };
const values = Object.values(person);
console.log(values); // Outputs: ["John", 30]
```

### 3. `.assign()`

The `Object.assign()` method copies all enumerable own properties from one or more source objects to a target object. It returns the target object.

#### Example:

```js showLineNumbers
const target = { a: 1 };
const source = { b: 2, c: 3 };
Object.assign(target, source);
console.log(target); // Outputs: { a: 1, b: 2, c: 3 }
```

### 4. `.freeze()`

The `Object.freeze()` method freezes an object, preventing new properties from being added to it and marking all existing properties as read-only.

#### Example:

```js showLineNumbers
const person = { name: "John", age: 30 };
Object.freeze(person);
person.age = 31;
console.log(person.age); // Outputs: 30 (age remains unchanged)
```

### 5. `.fromEntries()`

The `Object.fromEntries()` method transforms a list of key-value pairs into an object.

Example:

```js showLineNumbers
const entries = [
  ["name", "John"],
  ["age", 30],
];
const person = Object.fromEntries(entries);
console.log(person); // Outputs: { name: "John", age: 30 }
```

### 6. `.entries()`

The `Object.entries()` method returns an array of a given object's own enumerable string-keyed property [key, value] pairs.

#### Example:

```js showLineNumbers
const person = { name: "John", age: 30 };
const entries = Object.entries(person);
console.log(entries); // Outputs: [["name", "John"], ["age", 30]] --> outtups similar to [...arr.entries()]
```

### Using `for...of` with Objects

While `for...of` cannot be used directly on objects, it can be utilized in combination with `Object.entries()` to iterate over an objectâ€™s properties.

```js showLineNumbers
const person = {
  name: "John",
  age: 30,
};

for (const [key, value] of Object.entries(person)) {
  console.log(`${key}: ${value}`);
}
// Outputs:
// "name: John"
// "age: 30"
```
